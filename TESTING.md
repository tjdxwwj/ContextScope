## 启动测试

### 1. 启动前端开发服务器

```bash
cd D:\code\request-analyzer\frontend
npm run dev
```

访问：**http://localhost:5173**

### 2. 启动 OpenClaw Gateway

```bash
openclaw gateway start
```

API 端点：**http://localhost:18789/plugins/contextscope/api**

## 已知限制

1. **WebSocket** - 当前为模拟连接，需要后端实现真实 WebSocket
2. **性能数据** - 延迟数据需要后端添加 latency 字段
3. **会话对比** - 需要后端实现 `/api/session` 端点
4. **实时推送** - 需要后端实现 WebSocket 推送新请求
